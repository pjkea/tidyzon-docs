## Notes

- The dashboard **SP Live Locations** map calls `?type=providers&limit=5000` to load all provider pins at once. The `coordinates` field is the plotted point.
- `coordinates.lat` and `coordinates.lng` are `null` when the entity has never set a location (no GPS ping for providers, no saved address for users).
- `status` for providers reflects `isavailable`: `online` = 1, `offline` = 0.
- `verified` for providers is `true` only when BOTH `isbackgroundverified` AND `isequipmentverified` are true.
- User location entries do not include `rating`, `completedJobs`, `lastPing`, or `serviceType` — these fields are provider-only.
