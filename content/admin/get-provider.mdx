## Notes

- `is_background_verified` and `is_equipment_verified` are independent flags. The UI rolls them up: both true → Approved; both false → Unapproved; mixed → Pending Review.
- `recent_requests` is capped at 10 items. Full request history: `GET /v1/admin/requests?providerid={tidyspid}`.
- The `?userid=` query param is the calling admin's ID for audit logging — not the target provider.
