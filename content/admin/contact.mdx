## Notes

- This endpoint does not require an admin token — it uses the public admin route (`POST /v1/admin/contact`). In practice the route is still behind the API gateway's admin authorizer in the routes config.
- The email is sent from `tidyzon@gmail.com` (SES source) to `support@tidyzon.com`.
- All four fields must be non-empty strings; whitespace-only values are rejected.
