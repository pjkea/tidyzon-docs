# Admin API

The Admin endpoints allow privileged users to manage every aspect of the Tidyzon platform — users, providers, requests, payments, notifications, and more.

**Auth:** All admin endpoints require `Authorization: Bearer <id_token>` obtained from `POST /v1/admin/auth/login`.

---

## Get Admin Token

```bash
curl -s -X POST 'https://25p5ndzk5j.execute-api.us-east-1.amazonaws.com/dev/v1/admin/auth/login' \
  -H 'Content-Type: application/json' \
  -d '{"email":"your@email.com","password":"yourpassword"}' \
  | python3 -m json.tool
```

The `token.id_token` field is your Bearer token.

---

## Capabilities

| Area | Endpoints |
|------|-----------|
| **Users** | List, view, update customers |
| **Providers** | List, view provider details and onboarding |
| **Requests** | List, filter, view service requests and timelines |
| **Payments** | Transactions, statements |
| **Notifications** | Send, schedule, manage |
| **Service Types** | Create, update service products |
| **Locations** | Live map — providers and customers |
| **Permissions** | Role-based access control |
