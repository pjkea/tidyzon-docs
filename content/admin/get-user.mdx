## Notes

- `statistics.account_period` and `avg_rating` use the literal string `"new"` — check the type before numeric formatting.
- The `userid` query param visible on some calls (e.g. `?userid=9`) is the **calling admin's** user ID for audit logging; the path param `{userid}` is the **target user**.
- The Manage User page also calls `/v1/admin/requests?customerid={id}` and `/v1/admin/statements?userId={id}`.
